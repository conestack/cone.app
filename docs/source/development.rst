Development
===========

Install project::

    make install

Run python tests::

    make test

Run python tests with coverage::

    make coverage

Run JavaScript tests::

    make karma

Build JavaScript bundles::

    make rollup

Build CSS Styles::

    make scss

Extract i18n translation messages and build message catalogs::

    make lingua

Generate sphinx docs::

    make docs
