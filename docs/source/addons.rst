======
Addons
======

``cone.app`` can be extended with addon packages that provide additional
functionality. The following packages are available:


Core Packages
-------------

cone.tile
~~~~~~~~~

Tile-based component rendering system. Tiles are reusable UI components
that can be rendered independently and composed to build complex interfaces.

`PyPI <https://pypi.org/project/cone.tile/>`_ |
`GitHub <https://github.com/conestack/cone.tile>`_


User Management
---------------

cone.ugm
~~~~~~~~

User and Group Management UI for ``cone.app``. Provides administration
interfaces for managing users and groups.

`PyPI <https://pypi.org/project/cone.ugm/>`_ |
`GitHub <https://github.com/conestack/cone.ugm>`_


Backend Integrations
--------------------

cone.ldap
~~~~~~~~~

LDAP integration for ``cone.app``. Provides LDAP-based user and group
management backend using ``node.ext.ldap``.

`PyPI <https://pypi.org/project/cone.ldap/>`_ |
`GitHub <https://github.com/conestack/cone.ldap>`_

cone.sql
~~~~~~~~

SQL database integration for ``cone.app``. Provides SQLAlchemy-based
application model persistence.

`PyPI <https://pypi.org/project/cone.sql/>`_ |
`GitHub <https://github.com/conestack/cone.sql>`_

cone.zodb
~~~~~~~~~

ZODB integration for ``cone.app``. Provides ZODB-based application model
persistence using ``node.ext.zodb``.

`PyPI <https://pypi.org/project/cone.zodb/>`_ |
`GitHub <https://github.com/conestack/cone.zodb>`_


UI Components
-------------

cone.calendar
~~~~~~~~~~~~~

Calendar widget for ``cone.app`` based on FullCalendar. Provides calendar
views and event management functionality.

`PyPI <https://pypi.org/project/cone.calendar/>`_ |
`GitHub <https://github.com/conestack/cone.calendar>`_

cone.charts
~~~~~~~~~~~

Chart.js integration for ``cone.app``. Provides chart rendering capabilities
for data visualization.

`PyPI <https://pypi.org/project/cone.charts/>`_ |
`GitHub <https://github.com/conestack/cone.charts>`_

cone.fileupload
~~~~~~~~~~~~~~~

File upload widget for ``cone.app`` based on jQuery File Upload. Provides
drag-and-drop file upload functionality with progress indication.

`PyPI <https://pypi.org/project/cone.fileupload/>`_ |
`GitHub <https://github.com/conestack/cone.fileupload>`_

cone.maps
~~~~~~~~~

Leaflet maps integration for ``cone.app``. Provides interactive map
components with marker support, drawing tools, and various plugins.

`PyPI <https://pypi.org/project/cone.maps/>`_ |
`GitHub <https://github.com/conestack/cone.maps>`_

cone.three
~~~~~~~~~~

Three.js integration for ``cone.app``. Provides 3D rendering capabilities
using WebGL.

`PyPI <https://pypi.org/project/cone.three/>`_ |
`GitHub <https://github.com/conestack/cone.three>`_


Services
--------

cone.firebase
~~~~~~~~~~~~~

Firebase integration for ``cone.app``. Provides Firebase Cloud Messaging
support for push notifications.

`PyPI <https://pypi.org/project/cone.firebase/>`_ |
`GitHub <https://github.com/conestack/cone.firebase>`_

cone.tokens
~~~~~~~~~~~

Token management for ``cone.app``. Provides secure token generation and
validation functionality.

`PyPI <https://pypi.org/project/cone.tokens/>`_ |
`GitHub <https://github.com/conestack/cone.tokens>`_
