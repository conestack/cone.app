------------------

  translation factory is defined.
  [rnix, 2021-08-06]

  is defined.
  [rnix, 2021-08-06]

  [rnix, 2021-08-06]

  instance instead of using ``attr_value``. Lookup function is expected to be a

  [rnix, 2021-07-08]

  [rnix, 2021-06-30]
